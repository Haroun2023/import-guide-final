Comprendre le Code HS

Qu'est-ce que le Code HS ?

Le Code HS (Harmonized System) est un code numérique international qui classe les produits commerciaux. En Arabie Saoudite, le code est étendu à 8-12 chiffres pour plus de précision, contrairement aux 6 chiffres utilisés internationalement.

Pourquoi c'est important ?
  • Détermine les droits de douane applicables
  • Identifie les réglementations SASO applicables
  • Affecte la TVA et autres taxes
  • Utilisé pour les statistiques commerciales
Exemple Pratique
Vêtements en Coton pour Femmes
6204.62.00.10
6204 - Vêtements pour femmes
62 - Coton
00 - Catégorie standard
10 - Classification saoudienne spécifique
Implications
Droit de douane : 12%
TVA : 15%
Réglementations : Étiquetage SASO requis
